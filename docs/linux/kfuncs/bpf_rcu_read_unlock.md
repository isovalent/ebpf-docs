# KFunc `bpf_rcu_read_unlock`

## Definition

Release a RCU read lock region in the BPF program.

**Signature**

<!-- [KFUNC_DEF] -->
`#!c void bpf_rcu_read_unlock()`
<!-- [/KFUNC_DEF] -->

## Usage

See [`bpf_rcu_read_lock`](bpf_rcu_read_lock.md) for the usage of this kfunc.
