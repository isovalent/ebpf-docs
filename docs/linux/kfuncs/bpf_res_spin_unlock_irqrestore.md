# KFunc `bpf_res_spin_unlock_irqrestore`

<!-- [FEATURE_TAG](bpf_res_spin_unlock_irqrestore) -->
[:octicons-tag-24: v6.15](https://github.com/torvalds/linux/commit/97eb35f3ad42de1c932ef1f7e2f0044d4fca35f4)
<!-- [/FEATURE_TAG] -->

Release a Resilient Queued Spin Lock (`rqspinlock`).

## Definition

Releases the lock, restores Interrupt Request(IRQ) state from the stack and re-enables IRQs.

**Signature**

<!-- [KFUNC_DEF] -->
`#!c void bpf_res_spin_unlock_irqrestore(struct bpf_res_spin_lock *lock, long unsigned int *flags__irq_flag)`
<!-- [/KFUNC_DEF] -->

## Usage

See [`bpf_res_spin_lock`](bpf_res_spin_lock.md) for details.
