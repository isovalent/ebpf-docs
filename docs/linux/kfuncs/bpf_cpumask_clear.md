Clear all of the bits in a BPF CPU-mask.

## Definition

`cpumask`: The BPF CPU-mask being cleared.

**Signature**

<!-- [KFUNC_DEF] -->
`#!c void bpf_cpumask_clear(struct bpf_cpumask *cpumask)`
