# Helper function `bpf_probe_read`

## Defintion

> Copyright (c) 2015 The Libbpf Authors. All rights reserved.


<!-- [HELPER_FUNC_DEF] -->
For tracing programs, safely attempt to read _size_ bytes from kernel space address _unsafe_ptr_ and store the data in _dst_.

Generally, use **bpf_probe_read_user**() or **bpf_probe_read_kernel**() instead.

### Returns

0 on success, or a negative error in case of failure.

`#!c static long (*bpf_probe_read)(void *dst, __u32 size, const void *unsafe_ptr) = (void *) 4;`
