# Helper function `bpf_ktime_get_boot_ns`

<!-- [FEATURE_TAG](bpf_ktime_get_boot_ns) -->
[:octicons-tag-24: v5.8](https://github.com/torvalds/linux/commit/71d19214776e61b33da48f7c1b46e522c7f78221)

## Definition

> Copyright (c) 2015 The Libbpf Authors. All rights reserved.


<!-- [HELPER_FUNC_DEF] -->
Return the time elapsed since system boot, in nanoseconds. Does include the time the system was suspended. See: **clock_gettime**(**CLOCK_BOOTTIME**)

### Returns

Current _ktime_.

`#!c static __u64 (* const bpf_ktime_get_boot_ns)(void) = (void *) 125;`
<!-- [/HELPER_FUNC_DEF] -->
## Usage

Returns a 64-bit value representing the current kernel time in nanoseconds since the system boot, excluding any time the system was suspended. This can be useful for measuring time intervals or generating timestamps in eBPF programs.
