Open a BPF link from a pin path.

## Definition

`#!c struct bpf_link *bpf_link__open(const char *path);`

**Parameters**

- `path`: Path to the BPF link pin.

**Return**

- `struct bpf_link *`: Pointer to the opened BPF link. `NULL` on error.
